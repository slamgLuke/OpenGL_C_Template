# OpenGL Project for C

- Basic showcase of window making and importing GLSL files for shading.
- Makefile for building and running

## Dependencies

- OpenGL
- glew
- glfw3
- cglm

## Makefile usage

1. Build + Run (default)

```bash
$ make
```
or alternatively...
```bash
$ make buildrun
```

2. Build only

```bash
$ make build
```


3. Run only

```bash
$ make run
```
